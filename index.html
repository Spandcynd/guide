Лабораторна робота 1.0. Методичка

Java - мова що спочатку компілюється, а потім інтерпретується*, тому для запуску програми спочатку компілюємо її (команда javac) потім інтерпретуємо (команда java). Також буде використовуватися утиліта jar для архівування файлів. 

* - інтерпретатор, віртуальна машина

Основні команди

Основний параметр команди javac - це шлях до файлу вихідного тексту, тобто файлу з розширенням .java. В найпростішому випадку компіляція файла вихідного тексту робиться так:

javac Hello.java

Після компіляцій, в тій самій папці, що і файл .java, отримаємо відкомпільований файл Hello.class
Цей файл містить в собі класс з таким самим ім'ям, як і ім'я файлу без розширення. Щоб запустити цей класс, потрібно написати команду java, основний параметр якої це ім'я класу, який треба запустити.

java Hello

Однак, нам знадобляться додаткові параметри (прапорці) для цих команд, щоб отримати бажаний результат.

Додаткові параметри для команди javac
-d папка
-sourcepath папка
-cp папка

Прапорець -d (англ. destination, місцепризначення) приймає в якості параметра папку, в яку буде відкомпільовано файл(и) вихідного коду.
Прапорець -sourcepath приймає в якості параметра папку, де компілятору потрібно буде шукати файли вихідного коду для компіляції (у випадку, коли .java файлів декілька).
Прапорець -cp вказує де треба шукати вже відкомпільовані файли з розширенням .class (знадобиться в останньому завданні лабораторної роботи)

Додаткові параметри для команди java
-cp папка

Прапорець -cp приймає в якості параметра папку, в якій інтерпрататору джава треба шукати необхідний йому клас

Структура проєкту

Кожен проєкт буде мати приблизно таку структуру:

Lab1.0.1
classes
src

В src будуть зеберігатися файли вихідного коду .java, а в classes відкомпільовані файли .class

Всі команди треба виконувати з кореню проєкту, тобто папки Lab1.0.x
D:\learning\programs\Lab1.0\Lab1.0.1>javac -d cla...

Завдання

Завдання 1

Розробити консольну програму Hello у вигляді одного класу.
javac -d classes src\HelloWriter.java

Ця команда спочатку відкомпілює файл HelloWriter.java, який розміщується в папці src, і збереже результат компіляції в папці classes

Після виконаних команд структура проєкту буде виглядати так:

Lab1.0.1
classes
HelloWriter.class
src
HelloWriter.java

Після компіляцій запускаємо програму

java -cp classes HelloWriter

Завдання 2

Розробити консольну програму Hello у вигляді двох класів.
javac -d classes -sourcepath src src\HelloWriterTest.java
java -cp classes HelloWriterTest
